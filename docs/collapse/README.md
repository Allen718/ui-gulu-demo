# 手风琴组件
通过折叠面板收纳内容区域，支持同时展开多个面板或单个面板。
<collapse-demo></collapse-demo>
#### options
关于a-collapse
|参数|说明|类型|可选值|默认值
|------|---------------|---------|---------|---------|
|selected|默认打开项|Array|---| ---|
|single|多个面板还是单一面板|Boolean|false/true|true|
关于a-collapse-item
|参数|说明|类型|可选值|默认值
|------|---------------|---------|---------|---------|
|title|必传，每一项的标题|String|---| ---|
|name|必传，用于控制打开控制面板|String|'1','2','3'...|---
